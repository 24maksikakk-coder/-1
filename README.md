# 🧮 Калькулятор на 7 языках программирования

Простой консольный калькулятор, реализованный на 7 языках для демонстрации синтаксиса и ввода-вывода.

## Возможности

- Сложение, вычитание, умножение, деление
- Обработка деления на ноль
- Проверка некорректного оператора

## Запуск

### Python
```bash
cd 01-python
python calculator.py
